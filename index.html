The Ruby small text search engine.

What's that?

Picky is a really easy to use and fast Ruby search engine that helps your users find what they are looking for.

Try…

A simple library example (wait 3 seconds for it to boot up).

The examples section offers more.

What it is

A small-text search engine for categorized data, such as varchar fields from a database (such as most data).

It offers a HTTP interface, and a comfortable web front end for your application.

Continue reading in the in Detail section.

What it is not

A full-text search engine for uncategorized data (like text from a book). If you need that, look at Sphinx or Solr (Lucene). They will help.

Getting Started

Got 5 minutes? Install and run Picky on your computer and adapt the examples.
