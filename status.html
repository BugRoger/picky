Status and Contributions

Status

We have passed 1.3 - a live introspection interface - and are currently working towards 1.3.1+ releases, including string keys and a Redis backend.

Also, we are dabbling in the idea of offering tcp/socket interfaces, since quite a bit of time is lost in HTTP parsing, packaging.

Quite a few more features are planned.

See the Roadmap to see what's – probably – going to happen next.
